# USB_HISTORY_KILLER
This script is designed to search for infomation about the USB - devices and uninstall them.
Mainly designed for OS MCBC 3.0
Attribute:
 -c: Delete infomation the logs.
 -s: Search usb infomation logs.
 -h: Call of help.
